# UAC System Capacity & Care Load Analytics Dashboard

## 📌 Project Overview
This project analyzes system capacity and care load for unaccompanied children under the UAC program. It provides insights into intake, transfers, discharges, and overall system pressure using an interactive dashboard.

---

## 🎯 Objectives
- Analyze total system load across CBP and HHS
- Identify imbalance between inflow and outflow
- Detect backlog accumulation and stress periods
- Provide data-driven insights for decision-making

---

## 📊 Key Features
- 📈 Interactive dashboard using Streamlit
- 🔍 Date range filtering
- 📊 KPI metrics (Total Load, Net Intake, Backlog, Growth Rate)
- 📉 Trend analysis with graphs
- 📋 Detailed data view

---

## 🛠️ Technologies Used
- Python
- Pandas
- Streamlit
- Matplotlib
